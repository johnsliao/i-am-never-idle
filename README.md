# i-am-never-idle
Simple script to never be idle on Communicator/Lync/Skype

# Always appear available!  
## Moves mouse x+5, y+5 every 5 minutes  
## Closes in the background after 8 hours  

# Usage  
1.  Clone the repo  
2. `sh start.sh`  
3. Never appear idle again
